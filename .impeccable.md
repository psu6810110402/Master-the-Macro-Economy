# Impeccable Design Context

This file serves as the core instruction set for Impeccable AI design skills (`/audit`, `/polish`, `/delight`, etc.).

## Design Context

### Users
- **Primary Users**: Finance Students, Trainee Traders, and Game Players (Operatives).
- **Secondary Users**: Facilitators, Game Masters, and System Admins.
- **Context**: High-stakes, fast-paced macro-economic investment simulation. Users need to parse massive amounts of financial data at a glance.
- **Job to be done**: Making split-second trading decisions based on macro triggers (Black Swans, GDP, Interest Rates).
- **Emotions**: Intensity, Urgency, Professionalism, and High-Tech Competence.

### Brand Personality
- **Voice/Tone**: Authoritative, Clinical, Elite, and Mathematical.
- **3-Word Personality**: High-Tech, Institutional, Precision.
- **References**: Bloomberg Terminals, Matrix-like Cyberpunk, Elite Institutional Trading Platforms (Bridgewater, Citadel).
- **Anti-References**: Playful SaaS, Cute Mobile Games, B2C E-commerce. Do not use rounded corners larger than `sm` or `md`.

### Aesthetic Direction
- **Theme**: Pure Dark Mode ("Bloomberg-Editorial: High-Impact Financial Aesthetics").
- **Colors**:
  - Background: Deep Carbon Navy (`--bg-main`) / Slate Blue (`--bg-secondary`).
  - Text: Tinted White (`--text-primary`), never raw `#ffffff`.
  - Indicators: Acid Green (`--accent-up`), Cyber Red (`--accent-down`), Electric Indigo (`--accent-brand`).
- **Typography**:
  - Headers: `Outfit` (`--font-heading`) — Bold, tight tracking.
  - Body: `Murecho` (`--font-body`).
  - Numbers: Must always use `.tabular-nums`.

### Design Principles
1. **Data Density over Whitespace**: Pack information tightly but logically. Emphasize contrast over padding.
2. **Deterministic Motion**: Use exponential easing curves (`--ease-out-expo`); animations must feel crisp and purposeful, never floating or bouncy.
3. **High Contrast Signaling**: Market movements (Up/Down) must use strictly defined Acid Green and Cyber Red for immediate cognitive recognition.
4. **Institutional Utilitarianism**: Borders should be subtle (`--border-subtle`), with high-intensity focus states. Avoid decorative fluff.
